pyorica — Evaluation Framework

Offline benchmark workflow: a session file is replayed chunk-by-chunk through the same stateful pipeline used in live streaming, then post-hoc ICA analysis measures per-IC artifact energy reduction across all pipeline stages.

Design note: The offline ICA in ic_source_energy() is a separate measurement tool — it uses a stable MNE decomposition fitted on the IIR stage, not the online ORICA weights. This ensures the metric is independent of ORICA's convergence state and gives a reproducible ground-truth comparison across stages.
